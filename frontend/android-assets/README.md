# Android Assets

## App Icon
- Place your 1024x1024 PNG icon at `icon.png`
- Run `npx @capacitor/assets generate --android` to generate all icon sizes

## Splash Screen
- Place a 2732x2732 PNG at `splash.png`
- Use brand color #080B14 background with #00D4FF Neural Trading OS logo

## Generate all sizes
npx @capacitor/assets generate --android
